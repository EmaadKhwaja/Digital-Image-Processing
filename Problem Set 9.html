Problem 1: Phase-only Reconstruction¶

The initial guess is a matrix consisting of ones of the same shape as the phase matrix. By guessing a one-matrix, we restrict the algorithm to search for the magnitude of the image. This can be combined with the reference phase to reconstruct the original image.

The termination condition was set to $1 \times 10^{-10}.$ This small value reflects a small difference between the previous and current iteration, indicating a converging image.

Problem 2: Magnitude-only Reconstruction¶

I tried 3 initial conditions: a random complex matrix added to the provided magnitude data, a ones matrix, and a zeros matrix.

None of these produced results that looked like they converged to a real image in particular, although they are very different. The algorithm appears to be extremely dependent on initial conditions. It does not work well for magnitude-only images. This indicates that phase information is significantly more important than magnitude for image reconstruction.
